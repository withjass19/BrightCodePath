Inheritance

24. Write a Java program to create an abstract class Shape2D with abstract methods draw() and resize(). Create subclasses Rectangle and Circle that extend the Shape2D class and implement the respective methods to draw and resize each shape.

Code:

// Abstract class Shape2D
abstract class Shape2D {
    // Abstract methods
    public abstract void draw();
    public abstract void resize(double factor);
}
                                        
// Subclass Rectangle extending Shape2D
class Rectangle extends Shape2D {
    // Implementing the draw method for Rectangle
    
    public void draw() {
        System.out.println("Drawing a rectangle");
    }
                                        
    // Implementing the resize method for Rectangle
    
    public void resize(double factor) {
        System.out.println("Resizing the rectangle by a factor of " + factor);
    }
}
                                        
// Subclass Circle extending Shape2D
class Circle extends Shape2D {
    // Implementing the draw method for Circle
    
    public void draw() {
        System.out.println("Drawing a circle");
    }
                                        
    // Implementing the resize method for Circle
    
    public void resize(double factor) {
        System.out.println("Resizing the circle by a factor of " + factor);
    }
}
                                        
// Main class
public class AbstractShape2DExample {
    public static void main(String[] args) {
        // Create instances of Rectangle and Circle
        Rectangle rectangle = new Rectangle();
        Circle circle = new Circle();

        // Draw and resize the shapes
        System.out.println("Rectangle:");
        rectangle.draw();
        rectangle.resize(1.5);
                                        
        System.out.println(); // Empty line for clarity
                                        
        System.out.println("Circle:");
        circle.draw();
        circle.resize(2.0);
    }
}

Output:

Rectangle:
Drawing a rectangle
Resizing the rectangle by a factor of 1.5

Circle:
Drawing a circle
Resizing the circle by a factor of 2.0
